* HTML5
* CSS3

* Estructura semántica en HTML
* Creación de componentes tipo **card**
* Manejo del **Box Model**

Durante este proyecto reforcé conceptos importantes del desarrollo frontend:

* Cómo construir componentes visuales reutilizables
* Cómo organizar contenido dentro de una tarjeta
* Cómo mejorar la legibilidad mediante espaciado y tipografía
